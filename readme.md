

# Freemining React : Web client for Freemining

## Install
```bash
git clone github.com/monstermax/freemining-react
cd freemining-react
npm i
```

## Run (prod)
```bash
npm run prod
```


## Run (dev)
```bash
npm run dev
```


## Compile (dev to prod)
```bash
npm run prod
```
